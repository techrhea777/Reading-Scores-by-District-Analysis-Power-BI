**Student Performance Dashboard**
Overview

This project presents an interactive Power BI dashboard designed to analyze student performance across multiple districts, cities, and academic factors. The dashboard focuses on identifying trends in reading and math scores, helping stakeholders make data-driven educational decisions.

**Objectives**
Analyze average reading scores by city and district
Evaluate math performance trends over tutor periods
Compare performance across extracurricular activities
Track year-over-year improvements
Identify high-performing and underperforming regions
